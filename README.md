# Кейс эффективного выпускника

Интерактивный сайт школьного проекта о рациональном тайм-менеджменте для учащихся 10-11 классов при подготовке к экзаменам.

## Что внутри

- Помодоро-таймер для фокус-сессий.
- Динамический список задач на день.
- Блок "Лягушка дня" для самой сложной задачи.
- Матрица Эйзенхауэра для расстановки приоритетов.
- Краткие результаты исследования и выводы проекта.

Сайт работает как статический проект: достаточно открыть `index.html` в браузере.
